To run this project, please follow the following steps:
* Ensure you have XAMPP or any Web Server as well as Database server installed
* Clone the project from github on https://github.com/nicholasgabun/CS-506-ASSIGNMENT-2021.git
* Place the project folder in your Web server's public directory
* Import the database by running the "cshop.sql" file contained in the project folder
* visit the directory in your web browser.
